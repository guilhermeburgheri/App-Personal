# App Personal - Controle de Alunos

App Android feito em Flutter para controle de:
- Alunos
- Aulas
- Mensalidades (Pago/Pendente)

## Stack
- Flutter
- Isar (banco local)
- Android

## Funcionalidades
- CRUD de alunos
- Registro de aulas
- Controle de mensalidade por mês
- Relatório mensal de aulas
